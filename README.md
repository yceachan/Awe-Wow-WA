# Awe-WoW-WA：最好的魔兽WA入门指北

## Whoami

如是衣陈@Eachan，电子信息系本科在读，对魔兽的插件编程部分饶有兴趣。本人这里是一个WoW beginner和爱好者折腾WA和lua时留下的一些札记，仓库内容包括：

* lua快速会调库指北
* 魔兽lua api 相关，调试环境搭建
* lua、WA插件的官方仓库fork
* WA的一些亲历的bug，和官方的issue爬楼
* 各类光环制作原理及lua触发机制，官方wiki札记
* 宏机制相关
* 个人制作、收集的宏和WA仓库
